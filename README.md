# bushido-yandex-feed

YML feed and product exports for the Bushido catalog.

Yandex YML feed URL:

```text
https://raw.githubusercontent.com/dessmond29-star/bushido-yandex-feed/main/outputs/bushido2_yandex_yml.xml
```

- `outputs/bushido2_yandex_yml.xml` - YML feed for Yandex
- `outputs/bushido_yandex_yml.xml` - YML feed for Yandex
- `outputs/bushido_yandex_products.xlsx` - Excel product list
- `outputs/bushido_yandex_products.csv` - CSV product list
- `outputs/bushido_yandex_report.json` - generation report
